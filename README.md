# Pension-Calculator-Plus
Η αναβαθμισμένη και πλήρης έκδοση της εφαρμογής υπολογισμού ποσού σύνταξης.

![intro](./readme-assets/appIntro.gif)

# Σχετικά

Η εφαρμογή Pension Calculator Plus αναπτύχθηκε ως συνέχεια της αρχικής απλής έκδοσής της (Pension Calculator) η οποία υπολογίζει κατά προσέγγιση το ποσό της κύριας σύνταξης.
Η παρούσα έκδοση δίνει τη δυνατότητα να υπολογιστεί η κύρια σύνταξη (αναμένεται και η υλοποίηση υπολογισμού επικουρικής σύνταξης) με ακρίβεια, παρέχοντας τη δυνατότητα
να καταχωρισθούν η μηνιαίες αποδοχές των ασφαλισμένων για όλους τους φορείς ασφάλισης, συνυπολογίζοντας μάλιστα και τα παράλληλα διαστήματα και τους πλασματικούς χρόνους.

Η εφαρμογή βρίσκεται ακόμη σε αρκετά πρώιμο στάδιο, ωστόσο στο μέλλον θα υλοποιηθεί η πλήρης λειτουργικότητά της όσον αφορά τις κύριες και επικουρικές συντάξεις.

# Οδηγίες λειτουργίας

Η εφαρμογή χωρίζεται σε κεφάλαια, σε καθένα από τα οποία ο χρήστης καταχωρεί όλα τα απαιτούμενα στοιχεία. Στα κεφάλαια που αφορούν τις καταχωρήσεις αποδοχών, εφόσον δεν υπάρχει ασφάλιση 
σε κάποιον από τους φορείς που παρουσιάζονται ο χρήστης δε χρειάζεται να συμπληρώσει τίποτα και απλώς πατάει ΣΥΝΕΧΕΙΑ.

Στο τέλος πατώντας ΕΚΤΥΠΩΣΗ ΑΠΟΤΕΛΕΣΜΑΤΟΣ η εφαρμογή ζητά από τον χρήστη να αποθηκεύσει ένα αρχείο pdf όπου περιέχεται το αποτέλεσμα του αναλυτικού υπολογισμού.

# Οδηγίες εγκατάστασης

Η εφαρμογή υλοποιήθηκε σε περιβάλλον Qt Creator, με την πλατφόρμα Qt Quick Framework (C++), σε Fedora Linux. Για να μεταγλωτίσετε την εφαρμογή θα χρειαστείτε 
την εφαρμογή Qt Installer, η οποία είναι διαθέσιμη σε Windows, macOS και Linux στον ακόλουθο σύνδεσμο:

[Qt Open Source Binaries Installer](https://www.qt.io/development/download-qt-installer-oss)

Κατόπιν εκτελείτε την εγκατάσταση με βάσει τις οδηγίες που παρέχονται από τον installer, ωστόσο θα πρέπει να προσθέσετε κάποια extensions/additional libraries:

Extensions/Qt PDF
Additional Libraries/Qt Multimedia
Additional Libraries/Qt Quick 3D
Additional Libraries/Qt Quick Timeline
Additional Libraries/Qt Shader Tools
Build Tools/CMake
Build Tools/Ninja

Αν και συνήθως είναι προεπιλεγμένα από τον installer.

Η εφαρμογή Qt Installer θα εγκαταστήσει αυτομάτως το περιβάλλον Qt Creator και όλες τις απαραίτητες βιβλιοθήκες για να εκτελεστεί το πρόγραμμα.

Κατόπιν κατεβάζετε το repository είτε με την εντολή git clone είτε ως αρχείο zip, και το ανοίγετε στο Qt Creator, πατώντας configure και στη συέχεια run (Ctrl + R).
